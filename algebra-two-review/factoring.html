Factoring

First method: When "A" is equal to one.

Before we start you should know the standard format you'll use this method with is: ax^2+bx+c

How do we factor x^2-11+10?

Step 1: List A,B,C

A=1

B=-11

C=10

What two number that multiply to C and add to B?

-1*10=10

-1+-10=-11

So the answer is... (x-1)(1-10)

Another Example:


How do we factor x^2+14x+48?

Step 1: List A,B,C

A=1

B=14

C=48

Step 2: What two number that multiply to C and add to B?

6*18=48

6+8=14

So the answer is... (x+6)(x+8)

Second method: When "A" is more than one.(A.K.A "AC Method")


How do we factor 3p^2-2p-5?

Step 1: List A,B,C

A=3

B=-2

C=-5

Step 2: Multiply A & C

3 * -5= -15

Rewrite the equation: x^2 + bx + AC

p^2 - 2p - 15

Step 3: List A,B,C

A=1

B=-2

C=-15

Step 4: What two number that multiply to C and add to B?

3 * -5 = -15

3 + -5= -2

so we get... (p+3)(p-5)

but we're not done

Step 5: Add "a" back into the parenthesis

(3p+3)(3p-5)

Step 5: Divide if can by the greatest common denominator

(3p+3)/3 = (p+1)

So the answer is...(p+1)(3p-5)
